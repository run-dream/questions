- 原理

  - work dir
  - stage
  - history
  - HEAD

- 命令

  - git add / git stage 

    将work dir的变更提交到stage

  - git checkout

    将stage的内容还原到 work dir

  - git commit

    将stage的变更提交到history

  - git reset

    将文件从stage区移除

  - git checkout HEAD .

    将history的内容还原到 work dir
